# DOSW - Laboratorio 5

## Introducción

Este repositorio contiene la implementación del Laboratorio 5 para la asignatura Desarrollo de Software (DOSW). El objetivo del laboratorio es aplicar los conocimientos adquiridos en la materia para diseñar, implementar y entregar una pequeña aplicación Java basada en Maven, junto con la documentación y la estimación del trabajo realizada por el equipo.

## Contenido del repositorio

- Código fuente Java: `src/main/java`
- Recursos y configuración: `src/main/resources`
- Tests: `src/test/java`
- Archivo de build: `pom.xml`

## Cómo ejecutar

1. Compilar y ejecutar tests:

```
mvn clean test
```

2. Ejecutar la aplicación (desde la raíz del módulo):

```
mvn spring-boot:run
```

Requisitos: JDK 11+ y Maven instalados.

## Estimación por puntos de historia

1. Mayor dificultad a la hora de estimar:

    La principal dificultad fue la falta de experiencia del equipo, lo que provocó estimaciones imprecisas y diferencias en la percepción del esfuerzo.

2. ¿Fue fácil llegar a un consenso?:

    No fue inmediato; se discutieron varias rondas hasta alcanzar un acuerdo mediante la técnica de Planning Poker.

3. ¿Cómo resolvieron discrepancias en estimaciones?:

    Utilizamos Planning Poker: estimaciones individuales, discusión de diferencias y consenso final. En casos extremos se tomó un promedio razonado.
